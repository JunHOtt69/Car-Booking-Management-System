# 🚗 Car Booking Management System
 
A Java-based Car Booking Management System built with Object-Oriented Programming (OOP) principles. This system supports three types of users: **Admin**, **Salesman**, and **Customer**, each with their own dedicated interface and features.

## 📋 Features
 
### 🔑 Admin
- Manage cars (add, edit, delete)
- Manage users (customers & salesmen)
- Manage bookings
- View payment analysis
- View unpaid & cancelled bookings
- View feedback analysis
 
### 🧑‍💼 Salesman
- View and manage assigned cars
- View bookings
- View salesman profile
 
### 🙋 Customer
- Browse available cars
- Make, view, and cancel bookings
- Make payments
- Submit feedback
- View profile

## 🚀 Getting Started
 
### Prerequisites
- [Apache NetBeans IDE 25](https://netbeans.apache.org/front/main/index.html)
- JDK 8 or above
 
### Installation
 
1. **Clone the repository**
   ```bash
   git clone https://github.com/JunHOtt69/Car-Booking-Management-System.git
   ```
 
2. **Open the project in NetBeans**
   - Launch **NetBeans 25**
   - Go to **File > Open Project**
   - Navigate to the cloned `Car-Booking-Management-System` folder and select it
   - NetBeans will automatically recognize it as a project
 
3. **Build and Run**
   - Click the **▶ Run** button in the toolbar, or
   - Press **`F6`**
   - NetBeans will compile and launch the application automatically
